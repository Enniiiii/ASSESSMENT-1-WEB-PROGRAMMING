HTML AND CSS REFERENCE SHEET

This reference sheet will help you remember what certain tags and properties do.

HTML:

Body tag: What is written here has the content of the webpage.

Example: '< body > content content< /body >'

Link tag: This is how you link to a CSS file.

Example: '< link rel="stylesheet" href="style.css" type="text/css" >'

Div tag: This is how you create a division in the webpage.

Example: '< div > content content< /div >'

Paragraph tag: This is how you create a paragraph.

Example: '< p > content content< /p >'

Header tag: This is how you create a header.

Example: '< h1 > content content< /h1 >'

CSS:

Color: This is how you change the color of the text.

Example: 'color: red;'

Background color: This is how you change the background color.

Example: 'background-color: blue;'

Font size: This is how you change the font size.

Example: 'font-size: 20px;'
